 LAN + VSAT Communication System

This project demonstrates the design of a Local Area Network integrated with a VSAT (Very Small Aperture Terminal) satellite link for wide-area connectivity.

 Features
- LAN setup and subnetting
- VSAT uplink/downlink simulation
- Router configuration
- Basic network performance analysis

Technologies
- Cisco Packet Tracer
- Networking fundamentals
- Satellite communication principles

 Documentation
This repository contains:
- Network diagram (PNG)
- Configuration steps
- Simulation files 

 Use Case
Ideal for connecting remote locations requiring stable satellite connectivity.
